Perfect Properties

Problem Statement

Real estate investment is a growing trend when it comes to financial security, however, finding the right property can be a frustrating experience for buyers who are new to the market. Users need a simple way to search on the go, save information, and receive recommendations that are customized to their interests.

Possible solution

A new property website that is straightforward, easy to use, and includes useful features such as a recommendation, easy access to favorite listings and comparison betweent favorite properties which help user to make better dicision.
